# About this project

The posts project of MEAN stack course by Academind.

## Serve Angular

Use port 4400 to serve the project.
`ng serve --port 4400`

## Build Angular

Build Angular in production mode with AOT.
`ng build --prod --aot`

## Run Server

Use nodemon to run server-side app.
`nodemon server.js`
